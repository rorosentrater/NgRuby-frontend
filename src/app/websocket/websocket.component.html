Websocket Chat

This is a simple demonstration of how websockets can be used to make a chat app. This example operates on only 1 GLOBAL chat room and messages aren't stored in a database. Grab a friend or open this page in a second browser and start sending messages. A message log will be built and displayed below as messages are exchanged.
